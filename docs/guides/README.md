## Quick Tips

### For Beginners
1. Start with simple, focused prompts
2. Use built-in templates
3. Watch iterations in real-time with `--verbose`
4. Commit before and after loops
5. Don't worry if early iterations look messy

### For Intermediate Users
1. Create custom templates for your patterns
2. Use `--watch` mode for prompt development
3. Combine with testing (`--stop-on-test-failure`)
4. Experiment with different providers
5. Build your prompt library

### For Advanced Users
1. Use multi-repo orchestration
2. Integrate with CI/CD
3. Create custom convergence criteria
4. Share templates with your team
5. Contribute to the community
